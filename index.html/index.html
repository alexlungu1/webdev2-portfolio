INTRODUCTION

Welcome, this website might look like any other ordinary website but it’s different. This website has many different suggestions to show how social media can have many different purposes.

This website entails the many different social medias we use day to day and is shown in a new perspective. I want to show the good sides to social medias and how we can use it to our advantage as a society.

I want to give learning a new boost to students and how the apps they use daily can help them massively.

As I understand learning can be very hard and sometimes boring, I want to make it interesting and show students what simple tasks can help them.

This website is sectioned so you can read into the social media you use the most and what I think as a social media user myself are the most “popular”.

I wanted to show you a new perspective into each social media, for example like “Instagram” and how it has so many ideas and opportunities that could help your learning.

I want to show social media can help spark new ideas for the everyday student and how it can be beneficial to you in the simplest way.
